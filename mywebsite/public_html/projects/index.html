Book Bubbles






Summary

In this project I created a visualization of data from the website Goodreads using html, javascript, and a javascript library called d3.js. All of the data in this visualization is found using the goodreads API.

The primary goal of this visualization answers the question How can we visualize the way that two books are similar to each other?, and if you are really interested you can read about more of the details in this research paper, or look at the source code for yourself on my github.

Lazy Fridge







This is a web app that I created with a team of four during the synopsis 2017 hackathon. We had 42 hours to complete a project under the theme "optimization and automation" and Lazy fridge is what we came up with.

The idea behind lazy fridge is to automate the task of figuring out what to cook with the ingredients that you have available to you in your fridge. To make this as easy as possible, our goal was to have users take pictures of the contents of their fridge and recieve a list of recipies as an output. Do to time constraints we did not implement a camera function, but users can upload a link to a photo and still recieve recipies. This app was created using html, javascript, and lots of help from the microsoft cognative api and the fork to spoon api.

check out the source code on git hub
